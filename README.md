# AI-Powered Retail Superstore Dashboard

Advanced Business Intelligence Project using:

- Python
- Streamlit
- Prophet (Time Series Forecasting)
- Plotly

## Features

- Upload Superstore CSV file
- Region & Category Filters
- KPI Dashboard
- Monthly Sales Trend
- Profit Optimization
- 90-Day Sales Forecast

## Run Locally

pip install -r requirements.txt
streamlit run app.py
